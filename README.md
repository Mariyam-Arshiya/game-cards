 🃏 Game of Cards

A simple interactive card stack built using HTML and CSS.

Cards are stacked in the center with a fan effect.
Hover over any card to bring it to the front.

##🚀 How to Use

Download the project
Open index.html in your browser
Hover over the cards to see the effect ✨

🛠️ Tech Stack

HTML5
CSS3

✨ Features

Card stack with fan effect
Smooth hover animation
Pure CSS (no JavaScript)
